Help:
  • Use keyboard navigation keys.
  • Hit ONLY the red ball.
  • The score is evaluated using the target ball diameter and the total number of balls.
  • Challenge yourself by increasing the number of balls.
  • Create a preference link using these parameters: balls, dark_mode, bg_image, and xp_mode, example:
    https://mshwf.github.io/mates/?balls=20&bg_image=false&dark_mode=true&xp_mode=true
  • Shortcuts:
    • R to restart.
    • +/- to increase/decrease the number of balls.
    • Space to pause/resume.
